# Console Customer Manager

A simple C# console application for managing customers.
The project was built as a beginner practice project to review C# basics, OOP, methods, loops, lists, and LINQ.

## Features

* Add a new customer
* Show all customers
* Search for a customer by name
* Update customer information
* Delete a customer
* Simple console menu
* Customer data stored in memory using `List<Customer>`

## Technologies Used

* C#
* .NET Console App
* Object-Oriented Programming
* LINQ

## Project Structure

The application contains:

* `Customer` class
* Menu system
* Add customer method
* Show customers method
* Search customer method
* Update customer method
* Delete customer method

## Customer Model

Each customer has:

* `Id`
* `Name`
* `Phone`
